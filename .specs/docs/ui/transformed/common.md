# Common UI Components & Layouts

## Layout Patterns

### Screen Containers

- **Padding**: Horizontal padding of `24px` is standard across most screens.
- **Background**:
  - Default: `#FFFFFF` (Light) or `#0B3056` (Dark Blue/Secondary).
  - Attention/Warning: `#FF0000` or Gradient.

### Action Containers

- **BottomActionContainer**:
  - Padded container at the bottom of screens for primary actions.
  - Often has a background color (e.g., `#0B3056`) and rounded top corners (`radius_px: 40`).
  - Padding: `40px 24px 60px 24px`.

## Primitive Components

### Buttons

- **Primary Button**:
  - Height: `56px`
  - Radius: `28px` (Pill shape) or `12px` (Rounded Rect).
  - Background: `#FF6D00` (Primary) or `#0B3056` (Secondary).
  - Text: `#FFFFFF`, `16px`, weight `600`/`500`.
- **Secondary/Social Button**:

  - Height: `56px`
  - Radius: `28px`.
  - Background: `#FFFFFF`.
  - Text: `#000000`.

- **Icon Button**:
  - Used for Home screen controls.
  - Circle: `96x96` (Record).
  - Rounded Rect: `80x80` (Gallery, Menu).

### Typography

- **Headers**:

  - Font: `Audiowide` or `Inter` (Bold).
  - Sizes: `40px` (Logo), `28px` (Page Titles), `24px`.
  - Colors: `#FF6D00` (Primary), `#FFFFFF` (Inverse), `#000000`.

- **Body**:

  - Font: `Inter`.
  - Size: `16px`.
  - Colors: `#666666` (Secondary Text), `#FFFFFF` (Inverse).

- **Captions/Labels**:
  - Size: `14px` - `16px`.

### Visual Elements

- **Icons**:

  - Size: `24px` standard.
  - Colors: `#FFFFFF` or `#FF6D00`.

- **Dividers**:
  - Height: `2px`.
  - Color: `#FFFFFF`.
